## Project Overview
This project explores data locality optimization in high-performance computing (HPC). It demonstrates how memory access patterns affect performance using different data structures.

---

## Implementation
A Python-based experiment was implemented to compare:
- Dictionary (non-contiguous memory)
- List (contiguous memory)

The program measures execution time for summation operations across different dataset sizes.

---

## Results
- Lists performed significantly faster than dictionaries
- Performance improvements ranged from ~38% to over 50%
- Larger datasets showed greater performance benefits
